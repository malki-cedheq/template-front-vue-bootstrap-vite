## Testando a aplicação
 [1] Instalar dependênci
### Instalação de dependências

```sh
npm i
```

### Prettier Code Formatter

```sh
npm run format
```

### DEV: compilação e hot-reload desenvolvimento

```sh
npm run dev
```

### PROD: compilação minificada para produção

```sh
npm run build
```
